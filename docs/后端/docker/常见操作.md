## docker容器中安装curl等命令

查看容器的系统 : cat /etc/issue

1. 如果是Debian系统，会显示如下

Debian GNU/Linux 11 \n \l


这种情况使用apt-get安装

apt-get update
apt-get install curl
apt-get install wget

2. 如果是Alpine Linux

Welcome to Alpine Linux 3.5
Kernel \r on an \m (\l)

使用apk add安装

apk update
apk add curl
apk add vim
apk add tennet

## 容器内安装curl 
背景 ： docker node容器内 无法使用 apt-get 或者 yum 
apk add curl
