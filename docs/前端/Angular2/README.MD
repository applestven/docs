## 2.创建angular项目 
  (对应项目ng_Start)
 - angular cli
    + 安装脚手架 
       npm i -g @angular/cli
    + 创建项目 
        ng new project_name(项目名称)
    + 启动项目 
        cd project_name
        ng serve --open    
    + 创建组件 
        ng g c hello
